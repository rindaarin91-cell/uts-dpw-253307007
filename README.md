Project ini adalah sebuah website untuk Penerimaan Peserta Didik Baru (PPDB) di sekolah Aurora International School. Website ini bertujuan untuk memudahkan calon siswa dalam mendapatkan informasi pendaftaran dan mengisi formulir secara online.

Website ini memiliki tiga halaman utama yang saling terhubung:

1. Halaman Beranda : berisi sambutan dan informasi jadwal PPDB yang ditampilkan dalam bentuk tabel agar mudah dibaca.
2. Halaman Alur Pendaftaran : menjelaskan langkah-langkah pendaftaran secara urut dan dilengkapi ilustrasi supaya lebih mudah dipahami.
3. Halaman Formulir Pendaftaran : untuk calon siswa mengisi data diri secara online 

Dari segi tampilan, website ini menggunakan tema warna pink gradasi agar terlihat menarik dan konsisten sebagai identitas sekolah. Struktur halamanada beberapa bagian seperti header, navigasi, isi utama, dan footer. 

Untuk fitur interaktif, digunakan JavaScript seperti:

- Validasi form secara langsung untuk memastikan semua data terisi dengan benar
- Pengecekan NIK harus 16 digit angka
- Notifikasi yang muncul selama 3 detik di bagian bawah layar sebagai tanda bahwa aksi pengguna berhasil atau ada kesalahan

Secara keseluruhan, website ini dibuat agar mudah digunakan, informatif, dan menarik bagi calon siswa yang ingin mendaftar.
